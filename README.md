# Python Port Scanner

A TCP port scanner written in Python.

This tool scans ports **1–1024** on a target IP address or domain
and displays open ports.

Author: Harsh Mahajan  
Handle: Dankharsh

---

## Features

- Accepts IP address or domain
- Resolves domain to IP automatically
- Scans ports 1–1024
- Displays open ports only
- Simple command-line interface

---

## Usage

Run the script using Python:

python port_scanner.py

---

## Example

Enter IP or Domain: scanme.nmap.org

Scanning Target: 45.33.32.156

Port 22 OPEN  
Port 80 OPEN
